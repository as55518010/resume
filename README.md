## 項目介紹
### 預覽：[https://derrick.tk/curriculum_vitae/](https://derrick.tk/curriculum_vitae/)

##### 本項目旨在顯示個人履歷
### 特色
- vue3
- 響應式
- 自動生成 PDF

## 快速使用

1. fork本項目後再clone到本地修改
2. 進入項目目錄執行`yarn`安裝依賴
3. 執行`yarn dev`開始開發
4. 修改完後執行`yarn build`發布到ghpages

``` bash
# 安裝依賴包
yarn
# 啟動程序
yarn dev
# 打包上線
yarn build
```
